# Список задач для сервиса аналитики

## 1. Анализ транзакций
- [ ] Рассчитывать общую сумму транзакций за определенный период (день, неделя, месяц).
- [ ] Определять среднюю сумму транзакции.
- [ ] Выявлять самые частые категории транзакций.
- [ ] Находить аномальные транзакции (например, слишком крупные или нетипичные для клиента).

## 2. Анализ клиентов
- [ ] Сегментировать клиентов по различным критериям (например, по среднему балансу, количеству транзакций, дате регистрации).
- [ ] Рассчитывать "время жизни" клиента (customer lifetime value).
- [ ] Выявлять отток клиентов (churn rate).

## 3. Анализ кредитов
- [ ] Рассчитывать общий портфель активных кредитов.
- [ ] Анализировать распределение кредитов по статусам (активные, погашенные, просроченные).
- [ ] Рассчитывать среднюю процентную ставку по кредитам.
- [ ] Прогнозировать риски невозврата кредитов (кредитный скоринг).
- [ ] Сегментировать кредитный портфель (например, по типу клиентов, по срокам).

## 4. Анализ вкладов
- [ ] Рассчитывать общую сумму средств на вкладах.
- [ ] Анализировать динамику притока/оттока вкладов.
- [ ] Рассчитывать средний срок вклада.
- [ ] Прогнозировать вероятность закрытия вклада после окончания срока.
- [ ] Анализировать структуру вкладов (по срокам, по суммам).

## 5. Анализ по геолокации и выявление мошенничества
- [ ] Определять "домашний" регион клиента.
- [ ] Выявлять транзакции из нетипичных регионов.
- [ ] Анализировать логины из подозрительных IP-адресов.
- [ ] Создать систему оповещений о подозрительной активности.

## 6. Анализ использования карт
- [ ] Анализировать популярность использования карт.
- [ ] Отслеживать сроки действия карт для проактивного перевыпуска.
- [ ] Анализировать причины блокировки карт.

## 7. Операционная аналитика (банкоматы и отделения)
- [ ] Анализировать нагрузку на банкоматы и отделения.
- [ ] Оптимизировать расположение банкоматов.
- [ ] Оценивать эффективность работы отделений.

## 8. Анализ активности пользователей
- [ ] Анализировать частоту и время входа пользователей в систему.
- [ ] Выявлять неактивных пользователей.
- [ ] Анализировать используемые устройства для входа.

## 9. Отчеты
- [ ] Создавать ежедневные, еженедельные и ежемесячные отчеты по ключевым метрикам.
- [ ] Предоставлять возможность выгрузки отчетов в различных форматах (например, CSV, PDF).

## 10. Прогнозирование
- [ ] Прогнозировать объемы транзакций на будущие периоды.
- [ ] Прогнозировать возможный отток клиентов.

## 11. Интеграция
- [ ] Обеспечить интеграцию с другими сервисами (например, для получения данных о курсах валют).
